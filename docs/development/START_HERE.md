## ⚡ 3分钟快速启动

### 步骤 1: 安装依赖
```bash
npm run install-all
```

### 步骤 2: 设置数据库
```bash
# 创建数据库
createdb fft_solar_crm

# 导入表结构和示例数据
psql -d fft_solar_crm -f database/schema.sql
```

### 步骤 3: 配置环境变量
```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env，修改数据库密码
# DB_PASSWORD=你的PostgreSQL密码
```

### 步骤 4: 启动系统
```bash
npm run dev
```

### 步骤 5: 访问系统
```
🌐 打开浏览器: http://localhost:3000

👤 登录账号:
   用户名: admin
   密码: admin123
```

## 🔧 常见问题

### Q: 数据库连接失败？
A: 检查 PostgreSQL 是否运行，`.env` 中的密码是否正确

### Q: 端口被占用？
A: 修改 `.env` 中的 `PORT=5001` 改为其他端口

### Q: 前端无法连接后端？
A: 确保后端在 5000 端口运行，检查 `client/package.json` 的 proxy 配置

### Q: 如何重置数据库？
```bash
psql -d fft_solar_crm -c "DROP SCHEMA public CASCADE; CREATE SCHEMA public;"
psql -d fft_solar_crm -f database/schema.sql
```

## 🎓 业务逻辑

### 自动计算规则

**项目收入**
```
收入 = (板子瓦数 × 板子数量) × 甲方单价
```

**员工薪资**
```
领队/安装人员: 薪资 = 板子数量 × 单价
电工: 薪资 = 固定金额/项目
```

**项目利润**
```
利润 = 项目收入 - 所有员工薪资总和
```
