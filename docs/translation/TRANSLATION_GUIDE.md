## Translation Strategy

### Step 1: Find and Replace
Use your code editor's find-and-replace feature for common terms:
```javascript
// Example replacements
'员工管理' → 'Staff Management'
'甲方管理' → 'Client Management'
'项目管理' → 'Project Management'
'添加' → 'Add'
'编辑' → 'Edit'
'删除' → 'Delete'
'保存' → 'Save'
'取消' → 'Cancel'
```

### Step 2: Update Object Keys
```javascript
// Before
const roleNames = {
  leader: '领队',
  installer: '安装人员',
  electrician: '电工',
};

// After
const roleNames = {
  leader: 'Leader',
  installer: 'Installer',
  electrician: 'Electrician',
};
```

### Step 3: Update UI Text
```javascript
// Before
<Typography>员工管理</Typography>

// After
<Typography>Staff Management</Typography>
```

### Step 4: Test
- Rebuild frontend: `docker-compose build frontend`
- Restart: `docker-compose up -d`
- Check UI for any remaining Chinese text
